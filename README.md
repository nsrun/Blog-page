## 🛠️ Installation

Set up your environment with modern tools:

```bash
# Clone repository
git clone https://github.com/yourusername/Blog-Bridges-main.git
cd Blog-Bridges-main

# Use Poetry for dependency management
pip install poetry
poetry install

# Activate virtual environment
poetry shell
```

## 🚀 Usage

Start the application:

```bash
python main.py
```

Or, if using a development server:

```bash
uvicorn main:app --reload
```
